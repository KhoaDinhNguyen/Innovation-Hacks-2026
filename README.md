# Innovation-Hacks-2026

This repository now contains a React Native project built with Expo and TypeScript.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

3. Run the app on a target platform:

```bash
npm run android
npm run ios
npm run web
```

## Project structure

- `App.tsx` contains the starter screen.
- `app.json` holds Expo app configuration.
- `assets/` contains the default Expo images and icons.
